📉 O Gráfico que Salvou o Emprego de Carlos

No artigo anterior, Carlos apresentou ao CEO a média salarial de R$ 8.400 — e três meses depois a inadimplência disparou. Desta vez, Carlos foi chamado novamente. O CEO queria entender o que havia acontecido e Carlos sabia que nenhuma tabela de números ia convencer alguém que não é técnico.

Ele precisava de algo diferente: precisava mostrar o problema, não apenas calculá-lo. Foi aí que Carlos aprendeu que a escolha do gráfico certo pode ser a diferença entre uma decisão correta e um desastre.

Por Que Visualizar?

Números em tabelas exigem que o leitor faça o trabalho cognitivo de encontrar padrões. Gráficos fazem esse trabalho por ele. Mas o gráfico errado pode distorcer a realidade tanto quanto a média sozinha.

Histograma — A Distribuição Revelada

O histograma divide os dados em intervalos (bins) e mostra a frequência de cada um. É o gráfico mais importante para entender a forma de uma distribuição — assimetria, caudas, múltiplos picos.

Carlos plotou o histograma dos salários e mostrou ao CEO o que a média escondia:

A linha laranja é a média (R$ 8.4k), a linha azul é a mediana (R$ 3.5k). A cauda à direita — os clientes de alta renda — é o que inflou a média e mascarou o risco real da carteira.

⚠️ Cuidado com o número de bins Poucos bins escondem a forma real da distribuição, muitos bins criam ruído. Uma regra prática é usar a Regra de Sturges: \(k = 1 + \log_2(n)\), onde \(n\) é o número de observações.

Boxplot — Cinco Números em Um Gráfico

O boxplot resume a distribuição com cinco estatísticas: mínimo, Q1, mediana, Q3 e máximo — e ainda identifica outliers automaticamente pelo critério de Tukey (visto no Artigo 2).

É especialmente poderoso para comparar grupos. Carlos usou boxplots para comparar a distribuição de salários entre clientes adimplentes e inadimplentes:

Esquerda: clientes adimplentes (mediana R$ 4.2k).
Direita: inadimplentes (mediana R$ 2.1k).

🔬 Boxplot em Feature Engineering

Em pipelines de ML, boxplots são usados na fase de EDA (Análise Exploratória de Dados) para identificar features com alta concentração de outliers antes do treinamento. Uma feature com muitos outliers pode dominar modelos baseados em distância (KNN, SVM) e distorcer regressões lineares — o boxplot é o primeiro diagnóstico.

Gráfico de Barras — Comparação entre Categorias

Ideal para comparar valores entre categorias discretas. Carlos usou para mostrar a taxa de inadimplência por faixa de renda — o argumento definitivo para o CEO:

A taxa de inadimplência cai drasticamente conforme a renda aumenta. Isso justifica a estratificação do modelo de crédito por segmento — o insight que Carlos não conseguiu comunicar com a média.

💡 Barras vs. Pizza Evite gráficos de pizza — o olho humano é ruim em comparar ângulos. Barras horizontais são sempre mais legíveis para muitas categorias e permitem rótulos longos.

Para séries temporais, o gráfico de linha é o padrão. Carlos plotou a evolução da inadimplência mês a mês para mostrar quando o problema começou a se agravar:

A inadimplência começou a subir em março — exatamente quando a campanha de aquisição sem segmentação de renda foi lançada. O gráfico de linha tornou a causa-efeito visualmente óbvia.

Scatter Plot — Relação entre Variáveis

O gráfico de dispersão mostra a relação entre duas variáveis numéricas. É o ponto de partida para análise de correlação — tema do Artigo 6. Carlos usou para mostrar a relação entre renda e probabilidade de default:

Cada ponto é um cliente. A tendência descendente confirma: quanto maior a renda, menor a probabilidade de default. Mas note a dispersão — renda explica parte do risco, não tudo.

🔬 Scatter Plot e Correlação de Pearson

O scatter plot é a visualização da correlação. Quando os pontos formam uma linha ascendente, a correlação de Pearson \(r\) é positiva e próxima de 1. Quando descendente, negativa. Quando não há padrão, \(r \approx 0\). No Artigo 6, vamos formalizar isso matematicamente e ver quando Pearson falha — e por que Spearman é mais robusto para dados como renda.

Tabelas — Quando o Gráfico não Basta

Tabelas são superiores a gráficos quando o leitor precisa de valores exatos para tomar decisões ou quando há muitas dimensões simultâneas. A regra é: use gráfico para mostrar padrão, use tabela para mostrar precisão.

📝 Resumo para Levar na Mochila

  • Histograma para ver a forma da distribuição (assimetria, caudas, outliers).
  • Boxplot para comparar grupos e identificar outliers com o critério de Tukey.
  • Barras para comparar categorias; evite pizza sempre.
  • Linha para evolução temporal — torna causa-efeito visualmente óbvia.
  • Scatter para explorar relação entre duas variáveis numéricas.
  • Tabela quando o leitor precisa de valores exatos para decidir.
